# Mini Blog Explorer

## Project Title
Mini Blog Explorer

## What the App Does
Mini Blog Explorer is a React application that fetches users from an external API and displays them in a user-friendly interface. Users can:

- View blog posts
- Search users by title or body
- Filter posts by user
- View user details in a modal
- Refresh data from the API
- See loading and error states

---

## API Used

This project uses the JSONPlaceholder API:

https://jsonplaceholder.typicode.com/posts

The API provides sample users data used for developmemn training 
---

- Functional Components

## How to Run Locally

### 1. Clone the Repository

```bash
git clone https://github.com/namataj/Mini-blog-explorer.git

2.Navigate to the Project Folder
cd mini-blog-explorer

3.Install Dependencies
npm install

4.Start the development server
npm run dev
```

5.Open the application and run 
http://localhost:5175
